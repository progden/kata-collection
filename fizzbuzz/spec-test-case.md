# FizzBuzz Specification & Test Cases

## Rules

| 規則 | 條件 | 輸出 |
|------|------|------|
| 1 | 不是 3 或 5 的倍數 | 數字本身 |
| 2 | 數字是 3 的倍數 | `"Fizz"` |
| 3 | 數字是 5 的倍數 | `"Buzz"` |
| 4 | 數字同時是 3 和 5 的倍數 | `"FizzBuzz"` |

## Test Cases

| 輸入 | 輸出 | 測試規則 | 原因 |
|------|------|----------|------|
| 1 | `"1"` | Rule 1 | 不是 3 或 5 的倍數 |
| 2 | `"2"` | Rule 1 | 不是 3 或 5 的倍數 |
| 4 | `"4"` | Rule 1 | 不是 3 或 5 的倍數 |
| 3 | `"Fizz"` | Rule 2 | 3 的倍數 |
| 6 | `"Fizz"` | Rule 2 | 3 的倍數 (6 = 3×2) |
| 9 | `"Fizz"` | Rule 2 | 3 的倍數 (9 = 3×3) |
| 5 | `"Buzz"` | Rule 3 | 5 的倍數 |
| 10 | `"Buzz"` | Rule 3 | 5 的倍數 (10 = 5×2) |
| 15 | `"FizzBuzz"` | Rule 4 | 3 和 5 的倍數 (15 = 3×5) |
| 30 | `"FizzBuzz"` | Rule 4 | 3 和 5 的倍數 (30 = 3×5×2) |
